## 📚 知识点

### 1. 编程三要素

| 要素 | 说明 | 代码示例 |
|------|------|----------|
| 输入 | 用户给程序的信息 | `name = input("你的名字？")` |
| 处理 | 程序对信息进行处理 | `age = age + 1` |
| 输出 | 程序返回结果 | `print(age)` |

**理解方法：** 就像和对话，你说话（输入），我思考（处理），我回答（输出）

---

### 2. 变量

**定义：** 变量就是一个有名字的盒子，用来装东西。

**语法：**
```python
name = "老大"    # 把"老大"装进name这个盒子
age = 25         # 把25装进age这个盒子
```

**理解方法：**
- `=` 不是等于，是"把右边给左边"
- 想象：名字←"老大"，年龄←25

---

### 3. 数据类型

Python有四种基本数据类型：

| 类型 | 例子 | 说明 |
|------|------|------|
| 整数 | `18`, `100`, `-5` | 没小数点的数字 |
| 浮点数 | `3.14`, `0.5` | 有小数点的数字 |
| 字符串 | `"你好"`, `'Hello'` | 文字，用引号包起来 |
| 布尔值 | `True`, `False` | 对或错 |

**例子：**
```python
a = 100          # 整数
b = 3.14         # 浮点数
c = "Hello"      # 字符串
d = True         # 布尔值
```

---

### 4. 输入和输出

**输出：**
```python
print("你好")              # 输出文字
print(age)                 # 输出变量的值
print("我叫", name)        # 输出多个东西
```

**输入：**
```python
name = input("你叫什么名字？")  # 会问用户，等用户回答
print("你好，", name)            # 把用户的话输出出来
```

## ⚠️ 注意事项

### 1. 英文符号

Python只认识英文符号：

```python
✅ 正确：print("你好")   # 英文括号和引号
❌ 错误：print（"你好"）  # 中文括号
```

---

### 2. 引号的使用

字符串必须用引号包起来：

```python
✅ 正确：name = "老大"   # 用引号
❌ 错误：name = 老大     # 没有引号会报错
```

---

### 3. input()返回字符串

`input()` 得到的永远是字符串，如果要数字需要转换：

```python
age = input("你多大了？")    # 得到的是字符串 "25"
age = int(input("你多大了？"))  # 转换成整数 25
```
