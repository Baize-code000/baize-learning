## 📚 知识点

### 1. 条件判断是什么？

**定义：** 让程序根据不同情况做不同的事情。

**生活例子：**
- 如果下雨，就带伞；否则不带伞
- 如果饿了，就吃饭；否则不吃饭

**代码例子：**
```python
if 下雨了:
    带伞
else:
    不带伞
```

---

### 2. if 语法

**基本格式：**
```python
if 条件:
    如果条件成立，执行这里
```

**示例：**
```python
age = 18

if age >= 18:
    print("你已经成年了")
```

---

### 3. if-else 语法

**格式：**
```python
if 条件:
    如果条件成立，执行这里
else:
    如果条件不成立，执行这里
```

**示例：**
```python
age = 15

if age >= 18:
    print("你已经成年了")
else:
    print("你还是未成年人")
```

---

### 4. if-elif-else 语法

**格式：**
```python
if 条件1:
    如果条件1成立，执行这里
elif 条件2:
    如果条件1不成立但条件2成立，执行这里
else:
    如果以上条件都不成立，执行这里
```

**示例：**
```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

---

### 5. 用户输入 input()

**语法：**
```python
name = input("你叫什么名字？")
print("你好，", name)
```

**注意：**
- `input()` 返回的永远是字符串
- 如果要数字，需要转换：
  ```python
  age = int(input("你多大了？"))
  ```

## ⚠️ 注意事项

### 1. 缩进很重要

Python用缩进表示代码块：

```python
✅ 正确：
if age >= 18:
    print("成年了")  # 缩进4个空格

❌ 错误：
if age >= 18:
print("成年了")  # 没有缩进
```

---

### 2. 冒号不能忘

`if`、`elif`、`else` 后面都要有冒号：

```python
✅ 正确：if age >= 18:
❌ 错误：if age >= 18
```

---

### 3. 比较运算符

| 运算符 | 含义 | 例子 |
|--------|------|------|
| `==` | 等于 | `a == b` |
| `!=` | 不等于 | `a != b` |
| `>` | 大于 | `a > b` |
| `<` | 小于 | `a < b` |
| `>=` | 大于等于 | `a >= b` |
| `<=` | 小于等于 | `a <= b` |

**注意：** 判断相等用 `==`，不是 `=`

---

### 4. 字符串比较

比较字符串要用引号：

```python
✅ 正确：if mood == "好":
❌ 错误：if mood == 好:
```
